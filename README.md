# Xeyronox · Purple Hat Hacker Showcase

> A personal cybersecurity portfolio and knowledge-sharing platform built by **Xeyronox** — a purple hat hacker codename.

![Xeyronox](public/og-cover.png)

## 🔥 About

Xeyronox is a single-page showcase site for a purple hat hacker identity. It features studied security tools, ethical knowledge sharing, a Telegram learning channel, and a private project in development.

## 🛠 Tech Stack

- **React 18** + **TypeScript**
- **Vite** — fast dev server & build
- **Tailwind CSS** — utility-first styling
- **shadcn/ui** — accessible UI components
- **Framer Motion** — staggered entrance animations
- **Lucide React** — icon system

## 📂 Project Structure

```
src/
├── components/
│   ├── Navbar.tsx           # Fixed top navigation
│   ├── HeroSection.tsx      # Hero with codename intro
│   ├── StatsBar.tsx         # Animated stats strip
│   ├── TerminalSection.tsx  # Terminal-style bio
│   ├── ChannelSection.tsx   # Telegram channel CTA
│   ├── ProjectsSection.tsx  # Private project teaser
│   ├── ToolsSection.tsx     # 9 security tools grid
│   ├── ManifestoSection.tsx # Hacker manifesto
│   ├── CollectiveSection.tsx# 8-member collective grid
│   ├── CustomCursor.tsx     # Custom cursor effect
│   └── Footer.tsx           # Footer with socials
├── hooks/
│   └── useScrollReveal.ts   # Scroll-triggered animations
├── pages/
│   ├── Index.tsx            # Main landing page
│   └── NotFound.tsx         # 404 page
└── index.css                # Design tokens & global styles
```

## 🚀 Getting Started

```sh
# Clone the repo
git clone <YOUR_GIT_URL>

# Navigate to the project
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start dev server
npm run dev
```

## 📦 Build & Deploy

```sh
npm run build
```

Host the `dist/` folder on [Vercel](https://vercel.com) or any static hosting provider.
